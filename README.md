COVID-19 Interactive Dashboard


A modern, interactive COVID-19 dashboard showing global and country-specific data.

Project Overview

A responsive, dynamic web application that visualizes the latest COVID-19 statistics using real-time data from the disease.sh API
.

It displays key metrics such as total cases, deaths, and recoveries, and provides a daily cases chart to help users understand trends. Users can filter by country, switch between dark and light mode, and enjoy animated counters and skeleton loaders for a polished, professional experience.

Features

✅ Fetches live COVID-19 data from a reliable API

✅ Global & country-specific stats (cases, deaths, recovered)

✅ Interactive line chart of daily new cases (Chart.js)

✅ Animated counters for a modern feel

✅ Dark/light mode toggle

✅ Responsive cards with hover effects

✅ Skeleton loaders for smooth loading

✅ Error handling with friendly messages

✅ Last updated timestamp

Tech Stack

HTML5 & CSS3 – Semantic layout, responsive design, dark mode

JavaScript (ES6) – Fetch API, DOM manipulation, animated counters

Chart.js – Interactive line charts

Font Awesome – Icons for visual appeal

Public API – disease.sh
 for COVID-19 data
